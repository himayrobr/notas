primeros pasos

ejecutar comando en la consola

npm i

luego se debe crear el archivo .env

el cual debe contener esto:

DB_HOST=172.16.102.226
DB_PORT=3306
DB_USER=root
DB_PASSWORD=serganimonDzoro1
DB_NAME=hospital
EXPRESS_HOST_NAME=localhost
EXPRESS_PORT=5000

ahora el comando es:

npm run dev
